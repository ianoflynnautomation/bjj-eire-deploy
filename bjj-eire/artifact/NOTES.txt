To check rollout status:
  kubectl rollout status deployment/bjj-api -n {{ .Release.Namespace }}
  kubectl rollout status statefulset/bjj-mongodb -n {{ .Release.Namespace }}
  kubectl rollout status deployment/bjj-frontend -n {{ .Release.Namespace }}

For local access (after deploy.sh has set /etc/hosts):
  https://app.bjj.local   → Frontend
  https://api.bjj.local   → API

Upgrade with a new image tag:
  helm upgrade bjj-eire . \
    -f values.yaml -f values-prod.yaml \
    --set bjj-api.api.image.tag=v1.2.3 \
    --set bjj-frontend.frontend.image.tag=v1.2.3
